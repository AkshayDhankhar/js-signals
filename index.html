js-signals by millermedeiros

Custom Event/Messaging system for JavaScript inspired by AS3-Signals

A Signal is similar to a EventTarget/EventDispatcher or a pub/sub system, the main difference is that each event kind has it's own controller and doesn't rely on strings to call proper callbacks. Another advantage is that you can pass arbitrary parameters to callbacks and it also have some convenience methods that aren't present on other implementations of the *observer pattern*. This implementation is heavily inspired by Robert Penner's AS3-Signals but it has a different set of features/classes, the main target of this implementation is custom events and not replacing native DOM events.
